# Набор тест-кейсов: Нефункциональное тестирование

Набор содержит тест-кейсы для проверки производительности, совместимости, адаптивности, юзабилити, доступности, безопасности, локализации и работы в офлайн-режиме. Все проверки подтверждают качество атрибутов приложения, не связанных с бизнес-логикой.
