# Набор тест-кейсов: Дымовое тестирование (Smoke Test)

Минимальный набор критических проверок для подтверждения работоспособности основной функциональности приложения и пригодности сборки к дальнейшему тестированию. Этот набор содержит 8 ключевых сценариев, охватывающих основной цикл игры.

---

| Поле | Значение |
|------|---------|
| **Название** | Загрузка приложения и отображение главного меню |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Браузер готов, URL приложения доступен |
| **Шаги** | 1. Откройте приложение в браузере.<br>2. Дождитесь полной загрузки. |
| **Ожидаемый результат** | Страница открывается без ошибок. Отображается экран загрузки с анимацией. Затем появляется главное меню с кнопками выбора уровня. |

---

| Поле | Значение |
|------|---------|
| **Название** | Выбор уровня "Новичок" и начало игры |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Приложение загружено, главное меню видимо |
| **Шаги** | 1. Нажмите кнопку "Новичок".<br>2. Проверьте загрузку игрового поля. |
| **Ожидаемый результат** | Генерируется поле 8×8. Счётчик мин показывает "10". Счётчик времени показывает "00:00". Игра готова к первому ходу. |

| Поле | Значение |
|------|---------|
| **Название** | Первый клик по полю и начало игры |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Игра запущена на "Новичке", поле видимо |
| **Шаги** | 1. Выполните левый клик по произвольной клетке на поле.<br>2. Проверьте реакцию. |
| **Ожидаемый результат** | Клетка открывается (безопасная). Таймер начинает отсчёт (от 00:01). Счётчик ходов увеличивается. Если это пустая область, происходит рекурсивное открытие соседних клеток. |

---

| Поле | Значение |
|------|---------|
| **Название** | Установка и снятие флага правым кликом |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Игра запущена, произведён первый клик |
| **Шаги** | 1. Выполните правый клик по закрытой клетке.<br>2. Проверьте появление флага и счётчика.<br>3. Выполните повторный правый клик на том же клетке.<br>4. Проверьте удаление флага. |
| **Ожидаемый результат** | После первого клика: на клетке появляется флаг, счётчик флагов увеличивается (показывает, например, "1"). После второго клика: флаг исчезает, счётчик уменьшается (показывает "0"). |

| Поле | Значение |
|------|---------|
| **Название** | Поражение — открытие клетки с миной |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Игра запущена, произведены безопасные ходы |
| **Шаги** | 1. Откройте клетку с миной (методом подбора или предварительно определите её).<br>2. Проверьте реакцию. |
| **Ожидаемый результат** | Игра завершается поражением. Все мины становятся видимыми. Открытая мина выделена красным. Отображается экран результата "Вы проиграли!" с кнопками. Таймер останавливается. Игровое поле заблокировано. |

| Поле | Значение |
|------|---------|
| **Название** | Победа — открытие всех безопасных клеток |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Новая игра на "Новичке" |
| **Шаги** | 1. Используя DevTools или логику, откройте все 54 безопасные клетки (наиболее эффективный путь: начните с пустой области).<br>2. Проверьте реакцию при открытии последней безопасной клетки. |
| **Ожидаемый результат** | Игра завершается победой. Отображается экран "Вы победили!" с итоговым временем. Таймер останавливается. Кнопки для новой игры или возврата в меню появляются. |

| Поле | Значение |
|------|---------|
| **Название** | Кнопка "Новая игра" после завершения партии |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Игра завершена (победа или поражение), экран результата видим |
| **Шаги** | 1. Нажмите кнопку "Новая игра".<br>2. Проверьте генерацию нового поля. |
| **Ожидаемый результат** | Генерируется новое поле "Новичка" (8×8). Все счётчики сбрасываются (время 00:00, ходы 0, флаги 0). Игра готова к новому первому клику. |

| Поле | Значение |
|------|---------|
| **Название** | Сохранение и загрузка игры |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Игра запущена, произведены ходы, игра не завершена |
| **Шаги** | 1. Нажмите кнопку "Сохранить прогресс" (если видима) или используйте диалог сохранения из меню паузы.<br>2. Введите название (например, "Тест1").<br>3. Нажмите "Сохранить".<br>4. Вернитесь в главное меню ("В меню").<br>5. Откройте "Загрузить сохранение".<br>6. Выберите сохранение "Тест1".<br>7. Проверьте восстановление состояния. |
| **Ожидаемый результат** | Диалог сохранения открывается. Игра сохраняется с указанным именем. После загрузки: поле восстанавливается точно (все открытые и закрытые клетки, флаги), время и ходы восстанавливаются. Игра продолжается с сохранённого момента. |

| Поле | Значение |
|------|---------|
| **Название** | Отображение панели статистики |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Главное меню открыто, сыграны несколько игр |
| **Шаги** | 1. Нажмите кнопку "Статистика" в главном меню.<br>2. Проверьте отображение информации. |
| **Ожидаемый результат** | Панель статистики открывается. Отображаются: количество сыгранных игр, количество побед, количество поражений, процент побед. По уровнях показано количество побед и лучшее время. |
