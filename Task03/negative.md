# Набор тест-кейсов: Негативные сценарии функционального тестирования

Набор содержит тест-кейсы для проверки устойчивости игры к ошибкам пользователя, граничным случаям и некорректным действиям. Все сценарии подтверждают, что приложение корректно обрабатывает неправильные операции без сбоев.

---

## Некорректные действия с клетками

| Поле | Значение |
|------|---------|
| **Название** | Повторный клик по уже открытой клетке не вызывает изменений |
| **Тип тестирования** | Функциональное (негативное) |
| **Предусловия** | Игра запущена, некоторые клетки открыты |
| **Шаги** | 1. Выполните левый клик по открытой клетке.<br>2. Выполните второй левый клик по той же клетке.<br>3. Проверьте, изменилось ли состояние клетки. |
| **Ожидаемый результат** | Клетка не изменяется после повторного клика. Никаких побочных эффектов не возникает. Число/пусто остаётся прежним. |
